# Pizza Sales Analysis using SQL

## Project Overview

This project analyzes pizza sales data using SQL to identify revenue trends, customer preferences, and operational insights.

## Tools Used

- MySQL
- SQL
- Excel
- GitHub

## Business Questions Answered

1. Total Revenue
2. Total Orders
3. Average Order Value
4. Top 5 Pizzas by Revenue
5. Top 5 Pizzas by Quantity Sold
6. Revenue by Category
7. Revenue by Pizza Size
8. Peak Order Hours
9. Sales by Day of Week
10. Pizza Revenue Ranking

## SQL Concepts Used

- Joins
- Aggregate Functions
- CTEs
- Window Functions
- RANK()
- ROW_NUMBER()
- GROUP BY
- ORDER BY

## Key Insights

- Identified top-performing pizzas.
- Analyzed customer ordering patterns.
- Determined peak business hours.
- Evaluated category and size performance.

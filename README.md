# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes pizza sales data using **SQL** and visualizes key business insights through an interactive **Excel Dashboard**.

The objective was to identify sales trends, customer ordering behavior, top-performing products, and revenue-driving factors to support business decision-making.

---

## 🎯 Objectives

* Analyze overall business performance.
* Calculate key sales metrics.
* Identify top and bottom-selling pizzas.
* Determine peak ordering hours and busiest sales days.
* Analyze revenue contribution by pizza category and size.
* Build an interactive dashboard for data-driven insights.

Provides key business findings and recommendations.

---

## 🔑 Key Insights

* Total Revenue generated: ₹817,860+
* Average Order Value: ₹38.31
* Friday is the busiest sales day.
* Large-sized pizzas contribute the highest revenue.
* Classic category generates the most sales.
* Peak ordering occurs during lunch hours.
